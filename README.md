
# AI-Based Personalized Study Plan Generator

## Project Description
This project is an AI-based system that generates personalized study plans for students based on their academic performance.

## Features
- Predicts student performance using Machine Learning
- Generates customized study plans
- Simple and interactive web interface

## Technologies Used
- Python
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend)
- Logistic Regression (Machine Learning)

##  Workflow
1. User enters academic details
2. Data is sent to backend
3. Machine learning model predicts performance
4. Study plan is generated and displayed

##  How to Run
1. Install required libraries:
   python -m pip install pandas scikit-learn flask joblib numpy

2. Run the backend:
   python app.py

3. Open browser and go to:
   http://127.0.0.1:5000/
